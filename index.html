About Project |

This website is made for everyone who is curious to learn about Events done, (Satellites,Rockets) launched by NASA and also the ISS(Internatioal Space Station). This User Interface is easy to uderstandable by non-techincal guys and children, too. We have extracted infromation from the official websites on NASA and explained the details in a easy way with the help of Animations and 3D- Models. Quiz sections have been made on Each of the webpage related to their topics and If a user gets passing points in quiz they will be redirected to the page which is the fun part of project. A 3D-game of Rocket travelling from one celestial body to another (ex- Earth to Moon). The Game is quite interesting and you guys will love it.
